# Contributions

Thank you for your interest in contributing to the Data Platform.

## Development Setup

1. Clone the repository
2. Copy `.env.example` to `.env` and configure
3. Run `docker-compose up -d` to start infrastructure
4. Follow the setup instructions in README.md

## Code Style

- TypeScript: Follow the Next.js and React conventions used in the codebase
- Python: Follow PEP 8 with 100-character line limit
- Use meaningful variable and function names

## Testing

- Write tests for new functionality
- Ensure all tests pass before submitting changes
- Aim for >80% code coverage on critical paths

## Pull Requests

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Run tests
5. Submit a pull request

## Questions

For questions or discussions, please open an issue.
